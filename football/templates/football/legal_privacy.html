Soporte Términos {% if request.user.is_authenticated %} Cuenta {% endif %}

Esta política describe cómo tratamos la información necesaria para prestar el servicio 2J Football Intelligence (cuenta, datos deportivos y archivos que subas). No usamos tus datos para publicidad ni para seguimiento entre apps o sitios web de terceros.

Qué datos tratamos

  • Cuenta: usuario, email, credenciales de acceso.
  • Datos deportivos: clubes, equipos, jugadores, sesiones, partidos y estadísticas.
  • Archivos que subas: fotos y documentos (p. ej. licencias federativas).
  • Datos técnicos: logs y métricas para diagnóstico (sin incluir contraseñas).

Finalidad

  • Prestar el servicio y permitir el acceso a los módulos.
  • Generar informes/PDFs deportivos.
  • Mejorar la estabilidad y seguridad del sistema.

Conservación

Conservamos la información mientras exista la cuenta o sea necesario para la prestación del servicio. Puedes solicitar la baja desde la propia app.

Derechos

Puedes acceder, rectificar o solicitar la supresión de tus datos. Para incidencias o solicitudes, contacta en {{ support_email }}.
